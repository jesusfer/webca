How to generate a certificate request

With OpenSSL it's very easy to generate a key pair and a certificate request (CSR) that you can use to request certificates.

Steps

  1. Install OpenSSL: binaries

  2. Open a command prompt.

  3. To create a private key and certificate request, run the following command:

    openssl req -out request.csr -new -newkey rsa:2048 -keyout private.key

    This generates an encrypted private key and a public key in a certificate request. The key size is 2048 bits using the RSA algorithm.
    If you don't want to have a passphrase in your private key, use the-nodes parameter:

    openssl req -out request.csr -new -newkey rsa:2048 -nodes -keyout private.key

    The full documentation for the above command is:OpenSSL req

  4. The above command gives you two files:

    • private.key: your private key. Keep it to yourself.
    • request.csr: your public key in a certificate request. You can send this over to a Certification Authority to request a certificate.

Remember this

  • You should store your private key securely. If you lose it or it get's compromised, so does your certificate.
  • Use a passphrase to store to private key.
